analyzeContract

Analyzes a smart contract for security vulnerabilities.

Parameters

  • contractCode: The source code of the smart contract.
  • analyzer: The security analyzer to use.
  • config: The configuration options for the analyzer.

Returns

An AnalysisResult object indicating the analysis result.

Example

const result = await SecurityAnalysis.analyzeContract('pragma solidity ^0.8.0; contract MyContract { ... }', 'my-analyzer', { timeout: 30000 });

getAnalysisResult

Retrieves the analysis result for a previously analyzed contract.

Parameters

  • contractId: The ID of the analyzed contract.

Returns

An AnalysisResult object indicating the analysis result.

Example

const result = await SecurityAnalysis.getAnalysisResult('my-contract-id');

getSupportedAnalyzers

Retrieves a list of supported security analyzers.

Returns

An array of strings representing the supported analyzers.

Example

const analyzers = await SecurityAnalysis.getSupportedAnalyzers();

getAnalyzerConfig

Retrieves the configuration options for a specific analyzer.

Parameters

  • analyzer: The security analyzer to retrieve the configuration for.

Returns

An object containing the configuration options for the analyzer.

Example

const config = await SecurityAnalysis.getAnalyzerConfig('my-analyzer');

setAnalyzerConfig

Sets the configuration options for a specific analyzer.

Parameters

  • analyzer: The security analyzer to set the configuration for.
  • config: The configuration options to set.

Returns

A boolean indicating whether the configuration was successfully set.

Example

const success = await SecurityAnalysis.setAnalyzerConfig('my-analyzer', { timeout: 30000 });
