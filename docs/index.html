Formal Verifier

The Formal Verifier contract provides a way to formally verify the correctness of smart contracts using model checking and symbolic execution.

  • verifyContract
  • getVerificationResult

Security Analysis

The Security Analysis contract provides a way to analyze smart contracts for security vulnerabilities using static analysis and dynamic analysis.

  • analyzeContract
  • getAnalysisResult

Getting Started

To get started with the Formal Verifier and Security Analysis contracts, follow these steps:

  1. Install the contracts using npm or yarn.
  2. Import the contracts into your project.
  3. Use the contracts to verify and analyze your smart contracts.
