verifyContract

Verifies a smart contract using formal methods.

Parameters

  • contractCode: The source code of the smart contract.
  • language: The programming language of the smart contract.

Returns

A VerificationResult object indicating the verification status.

Example

const result = await FormalVerifier.verifyContract('pragma solidity ^0.8.0; contract MyContract { ... }', 'solidity');

getVerificationResult

Retrieves the verification result for a previously verified contract.

Parameters

  • contractId: The ID of the verified contract.

Returns

A VerificationResult object indicating the verification status.

Example

const result = await FormalVerifier.getVerificationResult('my-contract-id');

getSupportedLanguages

Retrieves a list of supported programming languages for formal verification.

Returns

An array of strings representing the supported languages.

Example

const languages = await FormalVerifier.getSupportedLanguages();
